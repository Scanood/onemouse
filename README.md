## 🍉Installation

1. 下载源代码：
```bash
git clone https://github.com/Scanood/onemouse.git
```
2. 安装依赖：
```bash
cd onemouse
npm i
```
3. 启动：
```bash
npm run start
```
更多设置请查阅：[Electron Forge](https://www.electronforge.io/)
